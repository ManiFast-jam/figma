# Figma Make Dosyalarını İndirme

Figma Make dosyalarını indirmek için aşağıdaki URI'ler kullanılabilir:

## Kaynak Dosyalar

Tüm kaynak dosyalar şu URI formatında:
- `file://figma/make/source/coejfjeUSUSYr9pGGmCCSi/[dosya-yolu]`

## Dosya Listesi

### Ana Dosyalar
- `App.tsx`
- `Attributions.md`
- `guidelines/Guidelines.md`

### Components

- `components/auth/*.tsx`
- `components/feed/*.tsx`
- `components/figma/*.tsx`
- `components/fintech/*.tsx`
- `components/layout/*.tsx`

### Data
